## ℹ️ Sobre o projeto

Landing page responsiva com seção de:

- Hero
- Sobre
- Beneficios
- Preços
- Galeria
- Contato

O projeto utiliza estrutura semantica em HTML e responsividade com CSS puro, com adaptacoes para desktop e mobile.

## 🧑‍💻 Tecnologias utilizadas

- HTML5
- CSS3
- Google Fonts
- Font Awesome

## ⬇️ Como executar localmente

1. Clone este repositorio.
2. Abra o arquivo `index.html` no navegador.

> Como alternativa, use a extensao Live Server na sua IDE para atualizacao automatica durante os testes.
